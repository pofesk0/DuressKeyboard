# DuressKeyboard

Это клавиатура для подготовки к экстренным ситуациям. Может стирать данные когда вы вводите специальный код. Работает только на экране блокировки, чтобы вы случайно не стёрли данные при обычном использовании. Поможет, если кто-то попытается заставить вас ввести пароль. В таком случае вы вводите код сброса вместо пароля и нажимаете стрелку Enter (⏎).

Вы устанавливаете свой код сброса заранее.

После включения этой клавиатуры удалите остальные клавиатуры и отключите системные через через adb shell pm disable-user --user 0 имя.пакета.программы. Если не знаете имя пакета системной клавиатуры или она даже не отображается в настройках приложений, используйте программу Package Manager (можно найти на F-droid). Если вы не отключите другие клавиатуры кроме данной, вас могут попытаться заставить переключиться на них перед вводом пароля.

Когда я делал это приложение, я смотрел на пример другого подобного с именем Duress. Его проблема была в том, что оно использовало спецвозможности для отслеживания ввода кода сброса, а их Android иногда автоматически отключает, даже если они были включены вручную. Клавиатура же 100% надежный вариант.

Приложение использует старый SDK 29, и это не случайность. На SDK 29 его функционал работает куда надежнее из-за меньших ограничений.
